# Historias de Usuario - Sistema de Biblioteca

## HU-01 Buscar libros en el catálogo

**Como** usuario de la biblioteca  
**Quiero** buscar libros por título, autor o categoría  
**Para** encontrar rápidamente el libro que necesito.

### Criterios de aceptación

- El sistema debe permitir buscar libros por título.
- El sistema debe permitir buscar libros por autor.
- El sistema debe permitir buscar libros por categoría.
- El sistema debe mostrar una lista de libros encontrados.
- Si no hay resultados, el sistema debe mostrar un mensaje indicando que no se encontraron libros.
